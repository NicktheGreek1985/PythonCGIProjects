Algernon Algorithms Module

By Nick Patrikeos

Divide & Conquer Algorithms

The following algorithms use the divide & conquer paradigm to solve their problem.

Karatsuba Integer Multiplication

Uses Karatsuba's Algorithm to multiply two integers of very large size.

Runs in O(n ** (log2 3)).

Inversion Count

Piggybacks on the mergesort algorithm to count the number of inversions in an array.

Runs in O(n log n).

Sorting Algorithms

The following algorithms work to solve the sorting problem deterministically.

MergeSort

Uses divide and conquer to solve sorting in O(n log n) time.

Runs in O(n log n).

Deterministic Quicksort

Sorts a list around a deterministically chosen pivot recursively.

Runs in O(n ** 2)

Randomised Algorithms

The following algorithms use randomisation to increase their probability of running in optimal time.

Randomised Quicksort

Sorts a list around a randomly chosen pivot recursively.

Runs in O(n log n) on average.

Karger's Contraction Algorithm

Uses randomisation to solve the minumum cut problem.

Runs in O(n ** 2).
